## 📘 项目简介

**校友世界** 源自曾经的 APP 项目 **啦特**(https://latter.cn)。  
啦特最初是一款界面风格类似 Twitter 的英文交流社区，但在寻找种子用户的过程中发现核心需求并不真实，因此项目方向被判断为伪需求。

基于已有的基础架构与经验，我们决定转向更加清晰、真实也更有长期价值的领域——**校友社区**。  
于是，**校友世界** 诞生了。

校友世界是一款开源、面向全球校友群体的社区类 APP，致力于为校友之间提供一个开放、友好、可持续的交流空间。

## 🛠️ 技术栈（可按需保留）
- **服务端**：Spring Boot + MySQL（使用 MyBatis）
- **客户端**：Flutter + SQLite
- **架构特点**：轻量、可扩展、支持全球用户
- **定位**：开源可再利用的校友社区基础框架
