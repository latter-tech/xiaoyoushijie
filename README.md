## 项目简介

本项目源自一个失败的APP项目——啦特(<a href="https://latter.cn">latter.cn</a>)，
啦特是一款在界面上模仿了推特的项目， 限制只能用英文交流， 
在寻找种子用户阶段发现这是一个伪产品需求， 
所以决定调整方向改为校友社区， 
于是就诞生了 校友世界。
